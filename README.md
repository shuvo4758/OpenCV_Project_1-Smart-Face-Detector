# OpenCV_Project_1-Smart-Face-Detector

--Overview--
This mini project is a real-time Face, Eye, and Smile detection system built using OpenCV. It uses Haar Cascade classifiers to detect faces from a webcam feed and then detects eyes and smiles within the detected face region.

--Project File--
smartFaceDetector.py
Python script that performs real-time face, eye, and smile detection using a webcam.

--Features--
Real-time video capture using webcam
Face detection using haarcascade_frontalface_default.xml
Eye detection using haarcascade_eye.xml
Smile detection using haarcascade_smile.xml
Displays detection messages on the video frame
Press 'q' to exit the application
